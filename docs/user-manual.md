## 1. はじめに

BT-SYS は仮想通貨の自動トレードシステムです。ダッシュボードからボットを作成し、bitFlyer 取引所での自動売買を管理できます。

**主な機能**

- bitFlyer での自動売買ボットの作成・管理
- MACD 戦略による売買シグナルの自動検出
- 損益・勝率・取引回数のリアルタイム表示
- ストップロスによるリスク管理

**動作環境**

- 推奨ブラウザ：Chrome / Safari / Firefox（最新版）
- スマートフォンからも利用可能

## 2. アカウント登録・ログイン

### 新規登録

1. `http://localhost:3000` にアクセス
2. ログイン画面の「アカウントを作成」をクリック
3. メールアドレスとパスワードを入力して「登録」

> パスワードは8文字以上を推奨します。

### ログイン

1. メールアドレスとパスワードを入力
2. 「ログイン」をクリック
3. ダッシュボードに遷移します

### ログアウト

左サイドバー下部の「ログアウト」をクリックします。

## 3. APIキーの登録

ボットを動かすには、bitFlyer の API キーが必要です。

### bitFlyer での API キー発行手順

1. [bitFlyer](https://bitflyer.com) にログイン
2. 「設定」→「API」→「新しいAPIキーを作成」
3. 必要な権限を付与：
   - **残高照会**（必須）
   - **取引**（自動売買に必須）
4. API Key と API Secret をコピーして保管

### BT-SYS への登録

1. 左サイドバーの「設定」をクリック
2. 「APIキーを追加」セクションに入力：
   - **取引所**：bitFlyer
   - **API Key**：bitFlyer で発行したAPIキー
   - **API Secret**：bitFlyer で発行したシークレット
3. 「APIキーを登録」をクリック
4. 接続確認が自動実行され、成功すると一覧に表示されます

> APIキーは暗号化して保存されます。シークレットが画面に表示されることはありません。

### APIキーの削除

登録済みキーの右端のゴミ箱アイコンをクリックすると削除できます。

## 4. ボットの作成

### 作成手順

1. ダッシュボード右上の「ボット追加」をクリック
2. 各項目を設定して「ボットを作成」をクリック

### 設定項目

#### 基本設定

| 項目 | 説明 | 例 |
|------|------|----|
| ボット名 | 管理用の名前（自由入力） | `BTC MACD Bot` |
| 取引所 | 取引先取引所 | `bitFlyer` |
| 通貨ペア | 売買する通貨ペア | `BTC/JPY` |
| トレード種別 | 現物 or 先物 | `現物` |

**通貨ペアの選択肢**

- `BTC/JPY`（ビットコイン）
- `ETH/JPY`（イーサリアム）
- `XRP/JPY`（リップル）

#### 資金・リスク管理

| 項目 | 説明 | 推奨値 |
|------|------|--------|
| 資金枠（円） | このボットが使う上限金額 | `10000` 〜 `100000` |
| ストップロス（%） | 含み損がこの%を超えたら自動決済 | `3` 〜 `10` |

> ストップロスは **ポジション単位** で適用されます。例：5% に設定した場合、買いポジションの含み損が5%を超えると自動的に売却します。

#### 戦略・実行設定

| 項目 | 説明 |
|------|------|
| 戦略 | 使用する売買戦略（現在は MACD のみ） |
| タイムフレーム | ローソク足の時間足 |
| 実行間隔（秒） | ボットが売買シグナルを確認する間隔 |

**タイムフレームと実行間隔の推奨組み合わせ**

| タイムフレーム | 実行間隔（秒） | 用途 |
|---------------|--------------|------|
| `1m`（1分足） | `60` | 高頻度・短期 |
| `5m`（5分足） | `300` | 短期 |
| `15m`（15分足） | `900` | 短期〜中期 |
| `1h`（1時間足） | `3600` | 中期（推奨） |
| `4h`（4時間足） | `14400` | 中長期 |
| `1d`（日足） | `86400` | 長期 |

> タイムフレームと実行間隔は揃えることを推奨します。例：1時間足を使うなら実行間隔は `3600` 秒（1時間）。

#### MACD パラメータ

MACD（Moving Average Convergence Divergence）は、2本の移動平均線の差を使うトレンドフォロー戦略です。

| パラメータ | 説明 | デフォルト |
|-----------|------|-----------|
| fast | 短期EMAの期間 | `12` |
| slow | 長期EMAの期間 | `26` |
| signal | シグナル線の期間 | `9` |

**売買ルール**
- **買い**：MACDラインがシグナルラインを上抜けしたとき
- **売り**：MACDラインがシグナルラインを下抜けしたとき

**設定例**

| 用途 | fast | slow | signal |
|------|------|------|--------|
| 標準（推奨） | `12` | `26` | `9` |
| 短期・積極的 | `5` | `13` | `5` |
| 長期・安定重視 | `20` | `50` | `9` |

## 5. ボットの操作（起動・停止・削除）

### 起動

ダッシュボードのボットカードにある「起動」ボタンをクリックします。  
ステータスが「稼働中」に変わります。

> 起動するには、対応する取引所のAPIキーが登録されている必要があります。

### 停止

「停止」ボタンをクリックします。  
ステータスが「停止」に変わります。  
**保有中のポジションは自動決済されません。** 手動または次の売りシグナルで決済されます。

### 削除

ボットカード右下のゴミ箱アイコンをクリックします。  
稼働中のボットは削除できません。先に停止してください。

### ステータスの種類

| ステータス | 意味 |
|-----------|------|
| 稼働中（緑） | ボットが実行中。シグナルを監視・自動売買している |
| 停止（グレー） | ボットが停止中 |
| エラー（赤） | 何らかのエラーが発生。ボットカードにエラー内容が表示される |

## 6. 成績の確認

ダッシュボードの各ボットカードに成績が表示されます。

| 指標 | 説明 |
|------|------|
| 損益 | 確定した利益・損失の合計（円）。プラスは緑、マイナスは赤 |
| 取引数 | 完了した売買の回数 |
| 勝率 | 利益が出た取引の割合（%） |

> 損益はポジションをクローズ（売却）した時点で確定します。保有中の含み損益は含まれません。

## 7. 設定パラメータ詳細

### ボットの実行タイミング

システムは60秒ごとに全ボットをチェックします。各ボットは設定した実行間隔（`interval_seconds`）が経過した場合のみ実行されます。

```
システムループ（60秒ごと）
  ├─ BotA: 実行間隔=3600秒 → 前回から3600秒経過？ → 実行 or スキップ
  ├─ BotB: 実行間隔=300秒  → 前回から300秒経過？  → 実行 or スキップ
  └─ BotC: 実行間隔=86400秒 → 前回から86400秒経過？ → 実行 or スキップ
```

### ストップロスの動作

1. ボットが買いポジションを保有中、毎回の実行時に損益率をチェック
2. 含み損が `stop_loss_pct` を超えた場合、即座に成行売りで決済
3. ボットは「エラー」ではなく「停止」状態になります

## 8. トラブルシューティング

### APIキー登録時に「Connection failed」が表示される

**原因：** APIキーが無効、またはbitFlyer上に存在しない。

**確認事項：**
1. bitFlyer のAPIキー管理画面でキーが存在するか確認
2. 「残高照会」権限が付与されているか確認
3. APIキー・シークレットのコピーミスがないか確認（前後のスペースに注意）
4. bitFlyer アカウントが本人確認済みか確認

### ボットが「エラー」になる

ボットカードにエラーメッセージが表示されます。主な原因：

| エラーメッセージ | 対処 |
|----------------|------|
| APIキーが見つからない | 設定画面でAPIキーを再登録 |
| 残高不足 | bitFlyer 口座の残高を確認 |
| 取引所への接続エラー | しばらく待ってから再起動 |

### ボットを起動しても取引が発生しない

以下を確認してください：
- 実行間隔の時間が経過するまで取引は発生しません（例：3600秒設定なら最大1時間待機）
- MACD のクロスオーバーが発生しない相場では取引シグナルが出ません
- 既にポジションを保有している場合、追加の買いシグナルは無視されます

### ログイン後にダッシュボードが表示されない

ブラウザのキャッシュをクリアして再度ログインしてください。それでも解決しない場合はページをリロード（Cmd+R / Ctrl+R）してください。
